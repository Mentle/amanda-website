📋 How to Add a New Project

  1. Click "Edit Portfolio Projects" above
  2. Click "Create new file"
  3. Name it: project-name.json
  4. Copy this template:
{
  "title": "Your Project Title",
  "description": "Project description here",
  "heroImage": "/images/uploads/hero-image.jpg",
  "projectImages": [
    "/images/uploads/image1.jpg",
    "/images/uploads/image2.jpg"
  ],
  "date": "2024-01-01",
  "featured": true
}

💡 Tip: All changes are automatically deployed to your website!
